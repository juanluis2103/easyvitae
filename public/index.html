Herramientas gratis para convertir Markdown y PDF

Esta es una suite de herramientas online y gratuitas para trabajar con Markdown (.md) y PDF. Convierte md a pdf, extrae el texto de un PDF a Markdown (pdf a md) y más, todo desde el navegador, sin instalar nada y sin registrarte.

Herramientas incluidas

  • Markdown a PDF: convierte tus archivos .md en PDFs limpios y seleccionables.
  • PDF a Markdown: sube un PDF, extrae su texto y obtén .md editable.
  • Markdown a DOCX y PDF a DOCX (próximamente).

Ideal para CVs, READMEs, documentación técnica, apuntes y cualquier documento en Markdown.
